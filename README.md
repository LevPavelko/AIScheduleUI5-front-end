# AIScheduleUI5-front-end

ui5schedule-app is a SAPUI5-based scheduling application that helps you display and manage calendar appointments directly in the browser created by AI. 


## 1.Clone the repository:

```bash
git clone <YOUR_REPO_URL> ui5schedule-app
cd ui5schedule-app
```

## 2.Install dependencies
```bash
npm install
```
## 3.Run
```bash
npm run start
```
